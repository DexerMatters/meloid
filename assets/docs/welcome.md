# Modes

You can press **[`]** (back tick) on your keyboard to switch *modes*.

- `Normal`: You can interact with the application with your mouse and keyboard.
- `Command`: You can input commands for a wider control of the player with Vim's style.

Hint: There is the third mode where you can freely customize the layout and widgets. You can definitely find it out.
